# Python Libraries Documentation Archive

This repository contains offline documentation archives for various Python libraries. Each documentation package is provided as a compressed `.rar` file for convenient download and storage.

## How to Use the Documentation Offline

1. Download the desired `.rar` file from this repository.
2. Extract the contents of the `.rar` archive to a local folder.
3. Open the `index.html` file inside the extracted folder.
4. Browse the full documentation locally in your web browser without requiring an internet connection.
